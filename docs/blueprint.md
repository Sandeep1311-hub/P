# **App Name**: Noirfolio

## Core Features:

- Project Showcase: Display featured projects with images, descriptions, and links.
- About Me Section: A personal introduction with a photo, bio, and contact information.
- Skills Section: Highlight your technical and soft skills with visual progress bars or icons.
- Contact Form: Allow visitors to send messages directly through the portfolio.

## Style Guidelines:

- Background color: Dark charcoal (#121212) for a sleek, modern feel.
- Primary color: Soft white (#F0F0F0) for text and main elements to ensure high contrast.
- Accent color: Deep violet (#7950F2) to highlight interactive elements and add a touch of sophistication.
- Body and headline font: 'Inter', a sans-serif, providing a clean, modern look for both headlines and body text.
- Use simple, minimalist icons in the primary color (#F0F0F0) to complement the dark theme.
- Employ a clean, grid-based layout to ensure content is well-organized and visually appealing.
- Use subtle transition effects to make the website more visually interesting, but keep it simple to maintain a professional feel.
